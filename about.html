UserID creation form

With the UserID creation form, you can request IT department to create a company userID for a new employee in three easy steps! It's possible to request a UserID for up to 5 employees at the same time.

  1. Navigate to the Create UserIDs -tab
  2. Add the new users information
  3. Click SUBMIT
